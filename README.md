# SceneViewExtension_RainEffects
SceneViewExtension_RainEffects is a technical research project focused on implementing efficient,<br>
high-fidelity rain and surface wetness effects using Unreal Engine's ISceneViewExtension (SVE) framework.<br>
The core objective of this project is to explore the architectural advantages of Scene View Extensions for environmental rendering. 

# Disclaimer (Read Before Use)
**Reference Only**<br>
The code in this repository is provided for reference purposes only and is NOT a plug-and-play solution.<br>
It will not function out-of-the-box in Unreal Engine as it depends on external private frameworks and specific project configurations.

**Experimental Nature**<br>
This is a Research & Development (R&D) artifact. It may contain experimental syntax, unoptimized paths, or non-standard practices intended for architectural exploration.

**Content Excluded**<br>
